# 四、基于数据集的训练流程验证（核心部分）

## 4.1 数据集说明

实验使用经过编码后的训练数据（pkl 文件），其中包含：

```
merged_features（特征）
false_positive（标签）
```

数据划分方式：

- 测试集（Test）
- 初始标注集（Labeled）
- 未标注数据池（Pool）

## 4.2 主动学习训练流程

基于已有 pkl 数据，训练流程如下：

```
1. 从 pkl 中加载数据
2. 划分 labeled / pool / test
3. 使用 labeled 数据训练初始模型
4. 在 pool 上进行预测
5. 根据不确定性采样选择样本
6. 将样本加入 labeled
7. 重新训练模型
8. 重复上述过程（多轮迭代）
```

## 4.3 训练过程验证

通过观察训练日志，验证：

- 每一轮训练均成功执行；
- 数据规模逐轮增加；
- 模型持续更新。

（插入 round 日志截图）
